About

d3.js is hard to comprehend in the beginning, at least it was for me. That's why there exists many libraries to make it easier. dc.js is one of them, but it does not work by itself, it works alongside Crossfilter. Crossfilter gives us linking for free while dc.js abstracts types of charts for us. If you want to draw a line chart, simply call dc.lineChart().

This tutorial will guide you through making two pie charts, a bar chart, and a area chart. To do this, we will be using the data set linked in the getting started section.

The visualization goal is to query if gender, martial status, age, and amount of given credit affect those who default.

Step 1: Cleaning up data

The data set that we are using is in xls format which is not something we want to work with. Therefore, we will need to convert it to something for JavaScript friendly, like JSON. I cleared the first row in Excel then converted the file into csv then imported it using Pandas. The data needs to be in records format; this step is not optional, the JavaScript we will write for Crossfilter requires this. The full code is below.

import pandas as pd
import json data = pd.read_csv('default of credit card clients.csv')
with open('data.json', 'w') as f:
    f.write(data.reset_index().to_json(orient='records'))
